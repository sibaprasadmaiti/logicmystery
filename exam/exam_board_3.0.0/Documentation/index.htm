Overview Exam Board - Online Exam Management System

Exam Board is a powerful management system online exams, provide innovative examination process and assessment solutions to Educational Institutions, Government/ PSU sector and leading corporate entities. With Exam Board, team of leaders & innovators is constantly researching on making examination process easy and simple.

 

Exam Board offer an integrated suite of web application to conduct online exam, online question bank, e-commerce & enrollment management. This system helps education group, academy, recruiter, teacher, management consulting, technology services and outsourcing company to conduct exam.

Features

    Complete Online Exam Management System
    User Management System
    Free and Paid Course Programs
    Unlimited Enrollment Package
    Paypal Payment Integrated
    Onetime/Recurring Membership System
    Auto Generate PDF Certificate

    Advanced Administration Panel

    Settings with Social and Google Analytics

Installation

Step 1 Upload theme files

Open the archive ExamBoard.zip to find the following resources:

There are 3 folders:

  1. Documentation
  2. Screenshot
  3. ExamBoard
Upload the content of the ExamBoard folder to your web server.

Step 2 Create Database

Create the database (use cpanel or whatever database management client you have available)

Step 3 Database Settings

Open database.php file and enter the database name, username and password you've choosen before

Step 4 Install

1. Extract the zip files to a folder on your computer. 1. Extract the archive to a folder on your computer
2. Upload the files to your webserver
3. Run the setup script at yourdomain.com/setup
4. If you have selected install demo data login with admin : 123456 If not login with the details you entered

Step 5 Remove Installation Directory

Delete folder install from your site

Admin Side Management

Administration Login

Point your browser to http://www.yourdomain.com/administration/index.php. You will find a admin login panel. Login here by your admin username and password.

Dashboard

After login to your admin panel you will find a Dashboard. Here you will find quick statistics about visitor traffic statistics, quck statistics and other menus. Described below:

Manage Users

Under the Users menu, you will find all of the users registered for your website. YOu can edit, delete or search any user from here.

Manage Courses or Subjects

Under the Subjects menu you will find all of the subjects or courses which you have created. You can add new course programs. You can edit or delete any subjects from here.

Manage Exams or Tests

Under Exams menu you will find all of the the exams or tests which you have created. You can add new exam under any subject or course programs. You can edit or delete any Exams from here.

Add Question and Answer

Under Exams menu you will find all of the exams and from the list action menu you can add question to any exams. On add quetion page you will have a text editor to write the quetion and on below you will have an option to add answer. You can add as much answer you want.

Manage Quetions

Under Questions menu you will find all of the questions you added on exams. From here you can edit or delete any questions and made the answer active or banned and correct the true or false answer.

Sales or Transaction

Under Dashboard menu you will find a option for sales or transaction. From here you can see the the course programs and amount has been purchased by the users and you can estimate the total income.

Website Settings

Under Settings menu you will find all of the settings option for your website.

Maintenance Mode

From settings page if you active the Maintenance mode then the visitor will find the below screen.

Website Admins

Under Admins menu from the right side you will find all of the admins the website already have and you can make necessary changes on them.

Add New Website Admin

Under Admins menu you will find Add Admin menu through which you can create new website admin.

User Side Management

Home Page

On the home page you will find a quick overview about the website, menu for login or registration, social icons with link and on footer you will find quick contact, facebook like stream, twitter feeds.

Course Page

On Course page you find all of the course programs avilable by the site. You can see the course details and fees. By clicking on the enroll button you can enroll on that course program and by clicking on exams button you will get all the exams available on this course program.

Enrollment Page

After login to this website you can access this page, from here you can enroll to any course programs.

Exams Page

On Exams page you will find all of the exams or tests available by the website. By clicking on Take button you can take any exams but not on the exams which you have not enrooled.

Account Details Page

On account details page user can edit their account details nad password.

Exam Record Page

On exam record page user will find all the exams taken by the user. User can retake any exams.

My Enrollment Page

On my enrollment page user user will find their enrolled course programs.

Get Certified Page

User can get their certificate for all passed exams they have taken by clicking the button this page.

PDF Certified

When User request for crtificate the user will get a pdf file with their account details and the marks statements.

Question Page

This is the single question view when user take any exams. User will complete exam by choosing answers and clicking on next question button, and see the result on last page.

Result Page

When user take any exams, after completing questions user will get result on this page.

Frequently Asked Questions

Website visitor can find quick "Frequently Asked Questions" page, where quick information have been posted, you can add new question and answer by editing the faq.php file.

Contact Page

User or website visitor can contact to the website administration through this page, that information will go to the site email address which were set on the administration settings page.

CSS Files and Structure

Styling in your way

Exam Board uses following css files for the front design
/css/bitset-framework.css
/css/elastislide.css
/css/flexslider.css

The bitset-framework.css and the style.css are the two main css file styling the website.

}
